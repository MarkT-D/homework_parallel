# Blockchain / Voting Contracts

- Ganache: local Ethereum test blockchain
- MetaMask: wallet & transaction signer
- Remix: Solidity IDE

Steps:
1. Start Ganache (Quickstart).
2. Import a Ganache private key into MetaMask.
3. Add Ganache as a custom network in MetaMask (RPC http://127.0.0.1:7545).
4. Load `VotingAssignment.sol` in Remix and compile/deploy.
